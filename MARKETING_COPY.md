I got tired of clicking recipe links and fighting through ads, popups, and 4-paragraph backstories to find the actual ingredients. So I built a tool that extracts just the core content from any URL.

Tech: Python/FastAPI backend with trafilatura + beautifulsoup4. Static frontend (no build step).
